# Jogo da Memória

Bem-vindo ao nosso jogo da memória! Feito com HTML, CSS e JavaScript, este passatempo desafia sua memória e entretém com uma simplicidade encantadora. O objetivo é simples: encontrar todos os pares de emojis correspondentes virando as cartas em pares.

## Como Jogar

1. Abra o arquivo `index.html` em seu navegador preferido.
2. Clique em duas cartas para revelar seus emojis.
3. Se os emojis combinarem, as cartas permanecerão viradas.
4. Se não combinarem, as cartas serão viradas novamente.
5. Repita o processo até encontrar todos os pares de emojis.

## Recursos

- Interface simples e intuitiva.
- Efeitos visuais agradáveis.
- Botão de reinício para começar um novo jogo a qualquer momento.

## Personalização

Sinta-se à vontade para personalizar o jogo! Altere os emojis na matriz `emojis` no arquivo `script.js` e ajuste o estilo no arquivo CSS conforme sua preferência. Divirta-se!
